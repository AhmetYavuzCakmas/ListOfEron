# ListOfEron

Kişi bilgilerini kaydetmek için Tkinter ile yapılmış basit bir masaüstü uygulaması.

## Özellikler

- ✅ İsim, numara, adres, alan, şehir ve açıklama bilgilerini kaydetme
- ✅ Enter tuşu ile form alanları arasında geçiş
- ✅ Bilgileri `data.txt` dosyasına otomatik kaydetme
- ✅ Eksik bilgi kontrolü
- ✅ Kullanıcı dostu arayüz

## Kurulum

1. Python 3.x yüklü olmalı
2. Gerekli kütüphaneler:
```bash
pip install tkinter
```

## Kullanım
```bash
python loe.py
```

## .exe Oluşturma
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --add-data "logo.png;." loe.py
```

.exe dosyası `dist` klasöründe oluşacaktır.
